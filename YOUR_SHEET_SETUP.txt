YOUR GOOGLE SHEETS SETUP - Quick Copy Template
================================================

STEP 1: Create Google Sheet
----------------------------
1. Go to: https://sheets.google.com/
2. Click "+ Blank" to create new sheet
3. Rename to: "Clinical Study Data"

---------------------------------
1. Rename "Sheet1" to: Submissions

   - Bold (Ctrl+B)
   - Background color: Blue (#4A90E2)
   - Text color: White
   - View ‚Üí Freeze ‚Üí 1 row

TROUBLESHOOTING
---------------
If you see "Range 'Submissions!A:A' not found":
‚Üí Make sure tab name is exactly "Submissions" (capital S, no spaces)

If you see "403 Permission denied":
‚Üí Go to Google Cloud Console ‚Üí OAuth consent screen
‚Üí Add your email as Test User

If authorization popup is blocked:
‚Üí Allow popups for localhost in browser settings

If you see "redirect_uri_mismatch":
‚Üí Go to Google Cloud Console ‚Üí Credentials
‚Üí Edit your OAuth Client ID
‚Üí Add "http://localhost:8000" to Authorized JavaScript origins
‚Üí Save and wait 5 minutes
